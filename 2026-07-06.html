Armin Ronacher — SOTA models can regress on tools
Better Models: Worse Tools
New
Jul 4, 2026 — lucumr.pocoo.org
Ronacher observed that his Pi coding tool's edit function was increasingly being called with extra, invented fields in the nested edits[] array — the edit content usually correct, but the arguments not matching the schema. The surprise: this is worse with newer Anthropic models — both Opus 4.8 and Sonnet 5 exhibit it, while none of the older models do. The takeaway is a caution against assuming monotonic progress: a frontier model can be worse at honoring a specific tool contract than its predecessor, so tool schemas and evals need to be re-checked with each model bump. Cross-referenced by Simon Willison the same day.

Notable Themes

Cross-cutting observations

  • Claude is now first-party on all three clouds — and the hardware story matters. GA on Azure Foundry atop NVIDIA GB300 Blackwell Ultra completes the AWS/Google/Azure trifecta and foregrounds inference economics (Microsoft's claimed ~40% faster TTFT). The competitive front is shifting from "which model" to "where it runs, how fast, and under what governance" — a continuation of last week's Enterprise-analytics/gateway push.
  • Progress isn't monotonic at the tool boundary. Armin Ronacher's finding that Opus 4.8 and Sonnet 5 mangle a tool schema that older models handled correctly is a pointed counter to "newer = strictly better." It argues for per-model tool evals and schema hardening on every upgrade — the practitioner counterpart to Anthropic's own eval-heavy release cadence.
  • The Fable pricing cliff is shaping real behavior. Simon Willison timed a whole sqlite-utils 4.0 push around Fable's Max-plan window closing "in a few days," and openly tallied the ~$149 token cost. Frontier-agent coding is now something people schedule and budget around, not just reach for — and the "use a cheaper model in a subagent" tip is becoming standard cost hygiene.
  • "Delegate the judgement, not just the task." Both Simon's AIE takeaways (let Fable decide when to test, and which model to use) and Karpathy's refreshed guidelines point the same way: the human's job is setting explicit assumptions and success criteria, then letting the agent make the meta-decisions. Orchestration is moving up a level of abstraction.
  • Cyber safety is being institutionalized, not just patched. Anthropic's Jul 2 jailbreak-severity framework — co-authored with Amazon, Microsoft, and Google — tries to turn the Fable/Mythos export-control episode into a shared standard, even as HN keeps surfacing concrete Claude Code attack surfaces (GitHub Action secrets, poisoned repos). Offense and standardized defense are advancing together.
