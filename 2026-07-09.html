GPT-5.6 (Sol / Terra / Luna) launches publicly today — government preview ends
Significant
Jul 9, 2026 — OpenAI / PYMNTS / Axios / CNBC
OpenAI is taking GPT-5.6 fully public today across ChatGPT, the API, and Codex, ending the 13-day, government-coordinated limited preview that began June 26. Per PYMNTS, the broad release follows the White House (via OSTP and the Office of the National Cyber Director) lifting the restriction request it had made after flagging Sol's cyber capability; OpenAI announced the plan in a Jul 8 X post and said it is already expanding preview access globally. The three models: Sol (flagship; strongest yet at agentic coding, biology, cybersecurity), Terra (~GPT-5.5 performance at half the cost), and Luna (cheapest/fastest). Reported pricing per Mtok: Sol $5/$30, Terra $2.50/$15, Luna $1/$6. Sol adds a max reasoning effort and an ultra mode that spins up subagents — reportedly lifting Terminal-Bench 2.1 from 88.8% (plain Sol) to 91.9%. Notably, this is the mirror image of Anthropic's Fable 5 arc: both frontier US models were gated on government cyber concerns, and both are now cleared within roughly a week of each other.

xAI / SpaceXAI — Grok 4.5 ships (V9, Cursor-trained)
Grok 4.5 released — "Opus-class," 1.5T-param V9, trained on Cursor session data
Significant
Jul 8, 2026 — x.ai / Axios / Roo's Newsletter
Musk's SpaceXAI shipped Grok 4.5 to the public — its new flagship, built on the 1.5-trillion-parameter V9 foundation (≈3× the prior V8-small) and, distinctively, supplemented with real developer-session data from Cursor (debugging traces, multi-file diffs, user corrections) following SpaceX's $60B agreement to acquire Cursor. Priced at $2/$6 per Mtok, ~80 tok/s, live now in Grok Build (default), inside Cursor on all plans, and the SpaceXAI console — not yet in the EU. Musk calls it "Opus-class, but faster, more token-efficient and lower cost." The token-efficiency claim is the real story: 15,954 output tokens per SWE-Bench Pro task vs 67,020 for Opus 4.8 — a 4.2× gap. On raw benchmarks it's more mixed: on xAI's own chart Grok 4.5 beats Opus 4.8 on DeepSWE 1.0 and Terminal-Bench 2.1 but loses on DeepSWE 1.1 and SWE-Bench Pro — and Claude Fable 5 leads all four. It also scores #1 on Harvey's Legal Agent Benchmark. All figures are vendor self-reported; no independent verification yet.
Cognition — SWE-1.7 (RL-on-RL coding model)
SWE-1.7 — agentic coding model on a Kimi K2.7 base, 1,000 tok/s via Cerebras
New
Jul 8, 2026 — cognition.com
Cognition released SWE-1.7, its most capable model yet, built for agentic software engineering via large-scale reinforcement learning on top of an already heavily-RL'd Kimi K2.7 Code base — available today in Devin at ~1,000 tokens/sec via Cerebras. Scores: Terminal-Bench 2.1 81.5% (vs GPT-5.5 84.2%, Opus 4.8 86.9%) and FrontierCode 1.1 42.3% (vs GPT-5.5 43.0%, Opus 4.8 46.5%) — near-frontier at a fraction of the cost. Cognition's framing is pointed: large gains from stacking their own RL on an already-RL'd open base challenge the idea of a "post-training ceiling." Like Grok 4.5 and Terra, the pitch is cost-performance, not benchmark leadership. Note the open-weights lineage — a Kimi (Moonshot) base underneath a US commercial coding model.

Notable Themes

Cross-cutting observations

  • The frontier-model logjam broke all at once. Within ~24 hours: GPT-5.6 cleared for public launch (White House lifted its cyber-restriction request), Grok 4.5 shipped, SWE-1.7 shipped, and GPT-Live shipped. Two of these (Fable 5, GPT-5.6) were gated on government cyber concerns and were cleared within a week of each other — the safety-eval machinery that made "in preview" the default state for the strongest models is now releasing that backlog in a burst.
  • The competitive axis has shifted from raw scores to tokens-per-task. Grok 4.5's marquee number isn't a benchmark win — it's using 15,954 output tokens where Opus 4.8 uses 67,020 (4.2×). Terra is "GPT-5.5 at half the cost." SWE-1.7 is "near-frontier at a fraction of the cost." With Fable 5 still topping the accuracy charts (per xAI's own chart), challengers are competing on cost-efficiency, not ceiling.
  • Coding models are being trained on real developer behavior and open bases. Grok 4.5 folded in actual Cursor session data (diffs, corrections, debugging traces); SWE-1.7 stacked RL on a Kimi K2.7 Code base and argues this breaks the "post-training ceiling." The training signal is moving from static code corpora toward live agentic sessions — and increasingly starts from open weights.
  • Google is the conspicuous absence. On the busiest launch day of the summer, every major lab except Google shipped or cleared a frontier model. Gemini 3.5 Pro is still previewing toward July 17 with no public benchmarks — leaving Google a spectator on a day defined by everyone else's releases.
  • Voice quietly rejoined the frontier. GPT-Live is the first substantive voice upgrade in a long time, and its architecture (a fast conversational model that delegates hard work to a frontier model in the background) mirrors the subagent/LLM-Council patterns showing up everywhere else — orchestration, not a single monolithic model, is becoming the default shape.
