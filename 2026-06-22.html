  • A genuinely quiet weekend. Four of the five tracked labs (Anthropic, OpenAI, AWS, DeepMind) published nothing new since Friday, and OpenAI's news index has now been static for three weeks. After a fortnight of Fable-5 drama and the Shazeer hire, the news flow has reset to practitioner tooling — a useful reminder that the daily cadence of "big lab announcements" is lumpy, not constant.
  • MCP's identity crisis: tool protocol or auth gateway? Sean Lynch's quote (via Willison) reframes the most valuable thing MCP does as moving credentials out of the agent's context window — echoing last week's "Zero-Touch OAuth for MCP," AWS AgentCore Gateway's inbound OAuth, and the new Agentic Resource Discovery spec. The center of gravity in agent infrastructure is shifting from "what tools can the model call?" to "how does the model authenticate safely without leaking secrets?"
  • Infrastructure is quietly becoming agent-shaped. Cloudflare's no-signup, 60-minute "temporary accounts" are pitched at agents but useful to everyone — the same pattern Willison flags repeatedly. The practical effect is that ephemeral, low-friction, credential-light deploy paths are becoming table stakes, whether or not the "for AI agents" label sticks.
  • The token-cost debate keeps surfacing skeptics. "The Token Compression Illusion" continues the thread that ran through O'Reilly's "The Subsidy Ended" and "Linear Thinking, Nonlinear Costs": as agent workloads scale, every claimed efficiency trick gets adversarial scrutiny, and the burden of proof is rising.
  • Open-source supply chain remains the soft underbelly. 10k Trojan-laden GitHub repos topping HN, alongside the standing "AI repos under pressure" thread, underscores that as more code (and more dependency installs) is generated and executed by agents, the trust boundary around public package and repo ecosystems is where the risk concentrates.
